# Legacy Root Archive

Este diretório guarda arquivos que estavam soltos na raiz e foram movidos para manter a estrutura principal limpa.

Objetivo:
- preservar histórico e referência
- evitar poluição visual na raiz
- reduzir risco de uso acidental de arquivos legados

Se algum desses arquivos voltar a ser usado em produção, mova de volta para o local funcional adequado e atualize os scripts correspondentes.
