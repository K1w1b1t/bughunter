# Daily Routine (Professional)

1. Scope sync
- Validate scope changes from each program.
- Refresh in-scope and out-of-scope lists.

2. Recon cycle
- Run subdomain and endpoint discovery.
- Detect new assets and stack changes.

3. Validation block
- Manually validate top signals only.
- Build minimal and reproducible PoC.

4. Reporting block
- Submit high-confidence findings first.
- Track triage feedback and adapt templates.

5. Metrics update
- Record time spent, accepted rate, duplicates, payout.
